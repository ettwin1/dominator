Regex Crossword

Can you solve a crossword puzzle, but with regular expressions? The website with these is here

  • [ONE]*[SKA]

    • This expression requires 0 or more instances of a O, N, or E, and then one instance of S, K, or A. Because there are only 3 slots, we know the first two are O, N, or E, and the last one is S, K, or A.

  • .*(RE|ER)

    • This expressions requires 0 or more instances of any character, then one instance of RE or ER. Because there are only 3 slots, we know the first slot is any character, and the last two are RE or ER.

  • A+[TUB]*

    • This expression requires 1 or more instance of A, and 0 or more instances of T, U, or B. We know the first slot is an A, but the next two could be A, T, U, or B.

  • (NF|FA|A|FN)+

    • This expression requires 1 or more instances of NF, FA, A, or FN. Since there are only 3 slots, we know A will be either the first slot or the last, or all three.

  • .*(A|E|I).*

    • This expression requires 0 or more of any character, one instance of A, E, or I, then 0 or more of any character. Because there are only three slots, A, E, or I will be the middle slot, and the other two could be any character.

  • [SUPER]*

    • This expression requires 0 or more instances of S, U, P, E, or R. Because there are three slots that need to have a letter, we know each slots is either S, U, P,E, or R.

  • [STUPA]+

    • This expressions requires 1 or more instances of S, T, U, P, or A. Because there are three slots that need to have a letter, we know each slot is either S, T, U, P, or A.

  • [^GOA]*

    • This expression requires 0 or more instances of any character except G, O, or A. Each slot in this row will not be a G, O, or A.

  • .*(O|S)*

    • This expressions requires 0 or more instances of any character, then 0 or more instances of O or S. Really, this expression won't require any specifications in this row.

  • [RUSH]+

    • This expression requires 1 or more instances of R, U, S, or H. So, each of the slots need to be R, U, S, or H.

  • (P|ET|O|TEA)*

    • This expression requires 0 or more instances of P, ET, O, or TEA. Since there are only three slots, there can only be one instance of TEA, or one instance of ET with an instance of P or O, or a mixture of Ps and Os.

  • .*[GAF]*

    • This Expressions requires 0 or more instances of any character, then 0 or more instances of G, A, or F. Really, this expression won't require any specifications in this row.

Using all of this information, and through deductive reasoning, you can find each one of the letters in the crossword. Here's the answer:

N O S

F E R

A T U
