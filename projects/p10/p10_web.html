How did I make this?

First, we add Bootstrap to our website. It's not a complete necessity, but it makes things easier for us to code.




In a JavaScript file, we can create a function that will create an HTML element. The element will have properties, defined in the text inputs, and we use the placeholder value of each input to determine what property to define.




Now we can make the functions that delete the last element and reset the input values.




These functions are used to hide/show the DOM-inator. When we hid the DOM-inator, it creates a button element that is used to show the DOM-inator again.
